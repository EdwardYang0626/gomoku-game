# 现代五子棋

一个使用现代网页技术开发的五子棋游戏。

## 特性

- 简约现代的界面设计
- 流畅的游戏体验
- 支持悔棋和重新开始
- 炫酷的胜利动画效果
- 响应式设计，支持移动设备

## 技术栈

- HTML5 Canvas
- 现代 CSS3
- 原生 JavaScript
